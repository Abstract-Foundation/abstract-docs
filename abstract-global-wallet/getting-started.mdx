Abstract Global Wallet (AGW) has two packages available for integration:

1. <Icon icon="react" /> [agw-react](/abstract-global-wallet/agw-react/overview):
   For frontend applications using React - built on [wagmi](https://github.com/wagmi-dev/wagmi).

2. <Icon icon="js" /> [agw-client](/abstract-global-wallet/agw-client/overview):
   For JavaScript/TypeScript environments - built on [viem](https://github.com/wagmi-dev/viem).

## Starter Kits

To kickstart a new project with AGW configured in a Next.js app, use our CLI tool:

```bash
npx @abstract-foundation/create-abstract-app@latest my-app
```

Or clone one of our [example repositories](https://github.com/Abstract-Foundation/examples)
to quickly get started:

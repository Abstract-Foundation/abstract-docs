Abstract Global Wallet (AGW) has two packages available for integration:

1. <Icon icon="js" /> [agw-client](/abstract-global-wallet/agw-client/getting-started):
   For JavaScript/TypeScript environments - built on [viem](https://github.com/wagmi-dev/viem).

2. <Icon icon="react" /> [agw-react](/abstract-global-wallet/agw-react/getting-started):
   For frontend applications using React - built on [wagmi](https://github.com/wagmi-dev/wagmi).

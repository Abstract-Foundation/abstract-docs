[Dynamic](https://www.dynamic.xyz/) enables you to add authentication and wallet functionality to your application. The `agw-react` package includes an option to include Abstract Global Wallet as a connection option in your Dynamic application.

<Card
  title="AGW + Dynamic Example Repo"

>
  Use our example repo to quickly get started with AGW and Dynamic.
</Card>

## Installation

Install the required dependencies:

```bash
npm install @abstract-foundation/agw-react @dynamic-labs/sdk-react
```

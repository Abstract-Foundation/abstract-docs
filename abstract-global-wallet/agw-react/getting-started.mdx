<Accordion title="Prerequisites">
  Ensure you have the following installed on your machine: 
  
- [Node.js](https://nodejs.org/en/download/) v18.0.0 or later.
- This page assumes you have already created a React project.

</Accordion>

## 1. Install agw-react

To install the `agw-react package`, run the following command:

```bash
npm install @abstract-foundation/agw-react
```

## 2. Setup the AbstractWalletProvider

Wrap your application in the `AbstractWalletProvider` component to enable the use of the package's hooks and components
throughout your application.

```tsx
import { AbstractWalletProvider } from "@abstract-wallet/agw-react";

const config = {
  testnet: true, // Required
  // Optionally, provide your own RPC URL (learn more: https://viem.sh/docs/clients/transports/http.html)
  // transport: http("https://your.abstract.node.example.com/rpc") // Optional
};

const App = () => {
  return (
    <AbstractWalletProvider config={config}>
      {/* Your application components */}
    </AbstractWalletProvider>
  );
};
```

## 3. Use React Hooks

With the provider setup, you can now use any of the available hooks from the agw-react package inside
any component within your application:

```tsx
import {
  useLoginWithAbstract,
  useGlobalWalletSignerAccount,
} from "@abstract-foundation/agw-react";

export default function SignIn() {
  // Prompt the user to sign in with AGW
  const { login, logout } = useLoginWithAbstract();
  // Get information about the user's wallet
  const { address, status } = useGlobalWalletSignerAccount();

  // Example UI:
  if (status === "connected") {
    return <div>Connected to {address}</div>;
  }

  return <button onClick={login}>Connect with AGW</button>;
}
```

## 4. Submit transactions

Once the user has signed in, use the client available via the `useGlobalWalletSignerClient` hook to submit transactions,
deploy smart contracts, and more.

```tsx
import { useGlobalWalletSignerClient } from "@abstract-foundation/agw-react";

export default function SendTransaction() {
  // Get the signer (the Abstract Global Wallet)
  const { data: client } = useGlobalWalletSignerClient();

  // Use the sigend-in account to submit a transaction:
  async function submitTransaction() {
    const hash = await client.sendTransaction({
      to: "0x273B3527BF5b607dE86F504fED49e1582dD2a1C6",
      data: "0x69",
    });
  }

  return <button onClick={submitTransaction}>Submit Transaction</button>;
}
```
